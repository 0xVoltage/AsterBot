How to Use AsterBot
Initial Configuration
  1. API Key: Enter your Aster Dex public key
  2. Secret Key: Enter your Aster Dex secret key
  3. Margin: Choose the maximum % per trade (10-50%)
  4. TP/SL: Configure default profit and loss
  5. Click Save Configuration
Operation
  1. Click Start Bot
  2. Monitor the real-time dashboard
  3. Monitor activity logs
  4. To stop, click Stop Bot
